latex
-----

   Copy these files to a location that is visible to LaTeX.
   i.e., visible on your TEXINPUTS environment variable.

python
------

   Copy these files to a location that is visible on your PYTHONPATH environment
   variable.

scripts
-------

   Copy these files to a location that is visible on your PATH environment
   variable.

   If necessary, make these scripts executable using

      chmod ugo+x *.sh
      chmod ugo+x *.py

The latest versions of latex/* and scripts/* can be found at

https://github.com/leo-brewin/hybrid-latex/tree/master/cadabra
